Portfolio

Melodic Mosaic

"Melodic Mosaic" is a dynamic web application for music lovers, curating an extensive collection of impactful albums across genres and eras. Its user-friendly interface facilitates easy exploration, featuring intuitive sorting, searching, and insightful album metadata. Emphasizing visual representation through album cover art, "Melodic Mosaic" offers a streamlined and interactive journey through the world of music.

Chrono Conquest

"Chrono Conquest: A Journey Through Time" is an immersive web-based Simon Says game where users embark on an engaging journey through time. The project showcases an exciting user interface with appealing colors and engaging visual effects. Players navigate through the game using game control buttons and are able to select their skill level, providing an individualized gaming experience. The project exhibits an innovative use of JavaScript to handle game mechanics, making the game interactive and dynamic. The clean design and smooth functionality of the game ensure a delightful user experience.

Nina's Menu

"Nina's Menu" is a comprehensive website for a Filipino restaurant, specializing in homemade Lumpia and other delightful Filipino dishes. The site incorporates a welcoming homepage, a detailed menu section featuring the restaurant's offerings, an engaging about page recounting the establishment's story, an easy-to-navigate ordering system, and comprehensive catering and contact sections. The project is adorned with a modern design and clean user interface, ensuring an excellent user experience. The website's aesthetic appeal is complemented by the use of Google Fonts such as Roboto and Pacifico, giving it a unique and modern touch.
